Privacy Policy

Last updated: July 23, 2026

CJP Bot Detector ("the extension") is built to work entirely on your device. This policy explains what data the extension handles and what never leaves your browser.

What the extension does

The extension scans the comments of YouTube videos you watch and labels comments from very new accounts as possible bots, using publicly available information (channel creation dates).

Data the extension stores — locally only

All of the following is kept in your browser's local extension storage (chrome.storage) on your device and is never transmitted to us or any third party:

  • Your settings: your YouTube Data API key, the account-age threshold, and the on/off toggle.
  • Detection records: publicly available data about YouTube channels whose comments were scanned — channel handle or ID, display name, public account creation date, and the IDs and titles of videos where their comments appeared.

You can export this data as JSON or permanently delete it at any time from the extension popup.

Data the extension does NOT collect

  • No personally identifiable information about you
  • No browsing history, clicks, or analytics of any kind
  • No account credentials — your API key is stored locally, used only to call Google's API, and never sent anywhere else
  • Nothing is sold, shared, or transferred to any third party

Network requests

To look up the public account-creation date of channels that commented, the extension makes requests to YouTube (https://www.youtube.com) — the same public endpoints youtube.com itself uses — or, if you provide an API key, to the official YouTube Data API v3 (https://www.googleapis.com). Both retrieve only public channel information (the "Joined" date). No request carries any personal data about you, and the YouTube requests are sent without your account credentials. These requests are governed by Google's Privacy Policy.

Remote code

The extension does not load or execute any remote code. All code ships inside the extension package reviewed by the Chrome Web Store.
